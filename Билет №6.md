# Билет №6: Оптимизация узлов интерполяции. Полиномы Чебышёва

**Содержание:**
1. Определение и рекуррентная формула.
2. Корни полинома Чебышёва.
3. Свойство наименьшего уклонения от нуля (минимизация погрешности).
4. Перенос узлов на произвольный отрезок $[a, b]$.

---

### 1. Полиномы Чебышёва: Определение

Пусть:
```math
T_n(x) = \cos(n \arccos x), \quad x \in [-1, 1]
```

**Вычислим первые полиномы:**
*   $n=0: \quad T_0(x) = \cos(0) = 1$
*   $n=1: \quad T_1(x) = \cos(\arccos x) = x$
*   $n=2: \quad T_2(x) = \cos(2 \arccos x) = 2\cos^2(\arccos x) - 1 = 2x^2 - 1$

**Рекуррентная формула:**
Обозначим $\varphi = \arccos x$. Тогда $T_n(x) = \cos(n\varphi)$.
Рассмотрим сумму:
```math
T_{n+1}(x) + T_{n-1}(x) = \cos((n+1)\varphi) + \cos((n-1)\varphi)
```
По формуле суммы косинусов ($\cos(\alpha+\beta) + \cos(\alpha-\beta) = 2\cos\alpha\cos\beta$):
```math
= 2 \cos(n\varphi) \cdot \cos(\varphi) = 2 T_n(x) \cdot x
```
Отсюда получаем рекуррентное соотношение:
```math
T_{n+1}(x) = 2x T_n(x) - T_{n-1}(x)
```

**Свойства:**
1.  Старший коэффициент при $x^n$ равен $2^{n-1}$ (для $n \ge 1$).
2.  Область определения и область значений на $[-1, 1]$: $|T_n(x)| \le 1$.
3.  Четность: если $n$ четное — функция четная, если $n$ нечетное — функция нечетная.
4.  На концах отрезка: $T_n(1) = 1$, $T_n(-1) = (-1)^n$.

---

### 2. Корни полинома Чебышёва

На отрезке $[-1, 1]$ полином $T_n(x)$ имеет ровно $n$ корней.
Решим уравнение $T_n(x) = 0$:
```math
\cos(n \arccos x) = 0
```
```math
n \arccos x = \frac{\pi}{2} + \pi k, \quad k \in \mathbb{Z}
```
```math
\arccos x_k = \frac{\pi}{2n} + \frac{\pi k}{n} = \frac{\pi(2k+1)}{2n}
```

Так как $0 \le \arccos x \le \pi$, подходят значения $k = 0, 1, \dots, n-1$.
**Формула корней:**
```math
x_k = \cos\left( \frac{\pi(2k+1)}{2n} \right), \quad k = 0, \dots, n-1
```
Если выбрать эти точки в качестве узлов интерполяции, приближение будет наилучшим.

### 3. Минимизация оценки погрешности интерполяции

Ошибка интерполяции оценивается как:
```math
|f(x) - P_{n-1}(x)| \le \frac{M_n}{n!} |\omega_n(x)|, \quad \text{где } \omega_n(x) = \prod_{i=1}^n (x-x_i)
```
Чтобы минимизировать ошибку, нужно минимизировать $\max_{[-1, 1]} |\omega_n(x)|$.

Заметим, что $\omega_n(x)$ — это полином степени $n$ со старшим коэффициентом 1 (так называемый *приведенный* или *усеченный* полином).

**Теорема о наилучшем приближении:**
Среди всех полиномов степени $n$ со старшим коэффициентом 1, наименьшее отклонение от нуля на отрезке $[-1, 1]$ имеет приведенный полином Чебышёва:
```math
\tilde{T}_n(x) = \frac{1}{2^{n-1}} T_n(x)
```

Его максимальное отклонение:
```math
\max_{x \in [-1, 1]} |\tilde{T}_n(x)| = \frac{1}{2^{n-1}} \max |T_n(x)| = \frac{1}{2^{n-1}}
```

Любой другой полином со старшим коэффициентом 1 будет иметь отклонение больше, чем $\frac{1}{2^{n-1}}$.

**Вывод:** В качестве узлов интерполяции $x_i$ выгодно брать корни полинома Чебышёва, тогда $\omega_n(x) \equiv \tilde{T}_n(x)$, и оценка погрешности становится минимально возможной:
```math
|e(x)| \le \frac{M_n}{n!} \cdot \frac{1}{2^{n-1}}
```

### 4. Перенос на произвольный отрезок $[a, b]$

Полиномы Чебышёва определены на $t \in [-1, 1]$. Если задача задана на $x \in [a, b]$, делаем линейную замену переменной.

Линейное отображение $[-1, 1] \to [a, b]$:
```math
-1 \to a, \quad 1 \to b
```
Система:
```math
\begin{cases} a = k(-1) + l \\ b = k(1) + l \end{cases} \Rightarrow \begin{cases} l = \frac{a+b}{2} \text{ (середина)} \\ k = \frac{b-a}{2} \text{ (полуразмах)} \end{cases}
```

Связь координат:
```math
x = \frac{a+b}{2} + \frac{b-a}{2}t
```

**Формула узлов для отрезка $[a, b]$:**
Берем корни $t_k$ на $[-1, 1]$ и подставляем в формулу:
```math
x_k = \frac{a+b}{2} + \frac{b-a}{2} \cos\left( \frac{\pi(2k+1)}{2n} \right), \quad k=0, \dots, n-1
```

При этом оценка максимума модуля полинома $\omega_n(x)$ на $[a, b]$ изменится с учетом масштабирования:
```math
\max |\omega_n(x)| = \frac{1}{2^{n-1}} \left( \frac{b-a}{2} \right)^n = 2 \left( \frac{b-a}{4} \right)^n
```
