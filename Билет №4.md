# Билет №4: Интерполяционный полином Ньютона

**Содержание:**
1. Разделенные разности (определение и свойства).
2. Интерполяционный полином Ньютона (построение).
3. Доказательство формулы.
4. Преимущества перед Лагранжем.
5. Пример расчета.

---

### 1. Разделенная разность

Определяется рекурсивно:

1.  **0-го порядка:** $f(x_1)$
2.  **1-го порядка:**
    ```math
f(x_1; x_2) = \frac{f(x_2) - f(x_1)}{x_2 - x_1}
```
3.  **2-го порядка:**
    ```math
f(x_1; x_2; x_3) = \frac{f(x_2; x_3) - f(x_1; x_2)}{x_3 - x_1}
```
    ...
4.  **$n$-го порядка (переход $n-1 \to n$):**
    ```math
f(x_1; \dots; x_{n+1}) = \frac{f(x_2; \dots; x_{n+1}) - f(x_1; \dots; x_n)}{x_{n+1} - x_1}
```

#### Теорема (явный вид разделенной разности)
```math
f(x_1; \dots; x_n) = \sum_{i=1}^n \frac{f(x_i)}{\prod_{\substack{j=1 \\ j \neq i}}^{n} (x_i - x_j)}
```

**Доказательство (методом математической индукции):**

**База индукции ($n=2$):**
```math
f(x_1; x_2) = \frac{f(x_2) - f(x_1)}{x_2 - x_1} = \frac{f(x_1)}{x_1 - x_2} + \frac{f(x_2)}{x_2 - x_1}
```
(Формула верна, т.к. $-(x_2-x_1) = x_1-x_2$).

**Предположение индукции:**
Пусть формула верна для всех $n \le k$.

**Шаг индукции:**
Рассмотрим для $k+1$ точек:
```math
f(x_1; \dots; x_{k+1}) = \frac{f(x_2; \dots; x_{k+1}) - f(x_1; \dots; x_k)}{x_{k+1} - x_1}
```
Подставим суммы по предположению индукции:
```math
= \frac{1}{x_{k+1} - x_1} \cdot \left( \sum_{i=2}^{k+1} \frac{f(x_i)}{\prod_{\substack{j=2 \\ j \neq i}}^{k+1} (x_i - x_j)} - \sum_{i=1}^{k} \frac{f(x_i)}{\prod_{\substack{j=1 \\ j \neq i}}^{k} (x_i - x_j)} \right)
```
После алгебраических преобразований (приведения слагаемых при одинаковых $f(x_i)$) получаем искомую сумму:
```math
= \sum_{i=1}^{k+1} \frac{f(x_i)}{\prod_{\substack{j=1 \\ j \neq i}}^{k+1} (x_i - x_j)}
```
*(Теорема доказана).*

---

### 2. Интерполяционный полином Ньютона

Даны точки $(x_1, y_1), \dots, (x_n, y_n)$.
Формула полинома:
```math
P_{n-1}(x) = f(x_1) + f(x_1; x_2)(x - x_1) + f(x_1; x_2; x_3)(x - x_1)(x - x_2) + \dots
```
```math
\dots + f(x_1; \dots; x_n) \prod_{j=1}^{n-1} (x - x_j)
```

**Преимущество:**
При добавлении новой точки $(x_{n+1}, y_{n+1})$ **не придётся пересчитывать весь полином**, достаточно добавить одно слагаемое:
```math
P_n(x) = P_{n-1}(x) + f(x_1; \dots; x_{n+1}) \cdot \prod_{j=1}^{n} (x - x_j)
```
В отличие от полинома Лагранжа, где нужно пересчитывать все базисные полиномы.

---

### 3. Доказательство корректности формулы Ньютона

**База индукции ($n=2$):**
Даны $(x_1, f(x_1)), (x_2, f(x_2))$.
```math
P_1(x) = f(x_1) + f(x_1; x_2)(x - x_1)
```
Проверка узлов:
1. $P_1(x_1) = f(x_1) + 0 = f(x_1)$ — верно.
2. $P_1(x_2) = f(x_1) + \frac{f(x_2)-f(x_1)}{x_2-x_1}(x_2-x_1) = f(x_2)$ — верно.

**Предположение индукции:**
Формула верна для $k$ точек ($n \le k$).
```math
P_{k-1}(x) = f(x_1) + \dots + f(x_1; \dots; x_k)(x-x_1)\dots(x-x_{k-1})
```

**Переход:** Добавлена точка $(x_{k+1}, f(x_{k+1}))$.
Строим $P_k(x) = P_{k-1}(x) + f(x_1; \dots; x_{k+1}) \prod_{j=1}^{k} (x - x_j)$.
Очевидно, что для $i = \overline{1, k}$: $P_k(x_i) = P_{k-1}(x_i) = f(x_i)$ (второе слагаемое обнуляется).

Надо доказать, что $P_k(x_{k+1}) = f(x_{k+1})$.
Введем вспомогательный полином $\tilde{P}_{k-1}(x)$, проходящий через точки $x_2, \dots, x_{k+1}$.
Рассмотрим разность $P_k(x) - \tilde{P}_{k-1}(x)$. Коэффициент при старшей степени $x^k$ у $P_k$ равен $f(x_1; \dots; x_{k+1})$.
Используя свойства разделенных разностей (рекуррентное соотношение), доказывается, что равенство выполняется.
$\Rightarrow P_k(x_{k+1}) = f(x_{k+1})$.

### 4. Пример (из лекции)

Точки: $(1, 2), (2, 9), (3, 28), (4, 65)$.

**Таблица разделенных разностей:**

1.  **1-й порядок:**
    *   $f(x_1; x_2) = \frac{9-2}{2-1} = 7$
    *   $f(x_2; x_3) = \frac{28-9}{3-2} = 19$
    *   $f(x_3; x_4) = \frac{65-28}{4-3} = 37$

2.  **2-й порядок:**
    *   $f(x_1; x_2; x_3) = \frac{19-7}{3-1} = \frac{12}{2} = 6$
    *   $f(x_2; x_3; x_4) = \frac{37-19}{4-2} = \frac{18}{2} = 9$

3.  **3-й порядок:**
    *   $f(x_1; x_2; x_3; x_4) = \frac{9-6}{4-1} = \frac{3}{3} = 1$

**Полином:**
```math
P_3(x) = 2 + 7(x-1) + 6(x-1)(x-2) + 1(x-1)(x-2)(x-3)
```
