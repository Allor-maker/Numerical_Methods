# Билет №18: Алгебраическая степень точности (АСТ)

**Содержание:**
1. Определение.
2. Условие (теорема) достижения заданной степени.
3. Пример (средняя точка).
4. Условия повышения степени (Эффект для четного числа шагов).

---

### 1. Определение

Будем говорить, что квадратурная формула имеет **алгебраическую степень точности $m$**, если она **точно** интегрирует все многочлены степени не более $m$, а некоторые многочлены степени $m+1$ уже нет.

---

### 2. Условие (теорема) заданной степени

**Утверждение:**
Для того чтобы квадратурная формула имела алгебраическую степень $m$, **необходимо и достаточно**, чтобы она точно интегрировала функции (базисные мономы): $1, x, x^2, \dots, x^m$.

**Доказательство:**

*   **Достаточность:**
    Пусть $P_m(x) = \alpha_0 + \alpha_1 x + \dots + \alpha_m x^m$.
    Рассмотрим квадратурную формулу (сумма взвешенных значений):
```math
(b-a) \sum_{k=0}^S w_k P_m(x_k) = (b-a) \sum_{k=0}^S w_k \sum_{r=0}^m \alpha_r x_k^r
```
    Поменяем порядок суммирования:
```math
= \sum_{r=0}^m \alpha_r \underbrace{\left( (b-a) \sum_{k=0}^S w_k x_k^r \right)}_{\text{КФ для монома } x^r}
```
    Так как по условию формула точна для мономов $x^r$, заменим сумму на интеграл:
```math
= \sum_{r=0}^m \alpha_r \int_a^b x^r dx = \int_a^b \left( \sum_{r=0}^m \alpha_r x^r \right) dx = \int_a^b P_m(x) dx
```
    $\Rightarrow$ Формула точна для любого полинома степени $m$.

*   **Необходимость:** Очевидна (так как мономы являются частным случаем полиномов).


---

### 3. Пример (Формула средней точки)

1.  Она точна для всех линейных функций ($\alpha x + \beta$):
```math
\int_a^b (\alpha x + \beta) dx = (b-a)\beta + \alpha \frac{b^2-a^2}{2} = (b-a)\left( \beta + \alpha \frac{a+b}{2} \right)
```
    Значение КФ: $(b-a) \cdot f(\frac{a+b}{2}) = (b-a)(\alpha \frac{a+b}{2} + \beta)$. (Совпадает).

2.  И не точна для $3x^2$ (параболы):
    Точный интеграл: $\int_a^b 3x^2 dx = b^3 - a^3$.
    По формуле: $(b-a) \cdot 3 \left( \frac{a+b}{2} \right)^2 = \frac{3}{4}(b-a)(a+b)^2$.
    Эти значения не равны: $b^3 - a^3 = (b-a)(b^2+ab+a^2) \neq \frac{3}{4}(b-a)(a^2+2ab+b^2)$.


---

### 4. Условия повышения степени точности

Рассмотрим интерполяционную квадратурную формулу $I[f] = \sum_{k=0}^S w_k f(x_k)$.
По определению, она точна для всех полиномов степени $S$.
Рассмотрим полином $P(x)$ степени **выше $S$**. Разделим его с остатком на узловой полином $\omega(x) = \prod_{k=0}^S (x - x_k)$:
```math
P(x) = V(x) \cdot \omega(x) + Z(x)
```
Где степень $Z(x) \le S$, значит, для него ошибка интегрирования $E[Z] = 0$.

Тогда ошибка интегрирования $P(x)$ равна:
```math
E[P] = E[V \cdot \omega] = \int_a^b V(x)\omega(x) dx - \underbrace{I[V \cdot \omega]}_{=0}
```
(Квадратурная сумма $I[V \cdot \omega]$ равна нулю, так как в узлах $x_k$ множитель $\omega(x_k) = 0$).

**Вывод:** Ошибка формулы равна интегралу:
```math
E[P] = \int_a^b V(x) \omega(x) dx
```

**Пример (Формула Симпсона):**
Узлы $a, \frac{a+b}{2}, b$. Узловой полином $\omega(x) = (x-a)(x-b)(x-\frac{a+b}{2})$.
Если мы возьмем полином степени $S+1 = 3$ (кубический), то при делении на $\omega(x)$ (степени 3) частное $V(x)$ будет константой ($V = \text{const}$).
Интеграл от произведения константы на $\omega(x)$ по симметричному отрезку:
```math
\int_a^b V \cdot (x-a)(x-b)(x-\frac{a+b}{2}) dx = 0
```
(Интеграл равен нулю из-за центральной симметрии графика $\omega(x)$ относительно середины отрезка).

$\Rightarrow$ Формула Симпсона точна для всех многочленов степени $S+1$ (то есть 3), хотя построена по 3 узлам (степень 2).

> **Причина повышения порядка:** Удачное (симметричное) расположение узлов квадратуры.

**Задача оптимизации (переход к Гауссу):**
Разместить узлы $x_k$ так, чтобы интеграл $\int_a^b V(x)\omega(x) dx$ обнулялся для полинома $V(x)$ как можно большей степени. Это приводит к квадратурам Гаусса.
