# Билет №2: Практический анализ корректности вычислительных задач

Исследование устойчивости к малым возмущениям на примерах нахождения членов арифметической и геометрической прогрессий, а также нахождения кратного корня полинома.

---

### 1. Нахождение члена арифметической прогрессии

**Дано:**
Формула $n$-го члена:
```math
a_n = a_1 + (n-1)d
```

**Анализ устойчивости:**
Пусть мы внесли возмущение в первый член $a_1$:
```math
\tilde{a}_1 = a_1 + \delta, \quad \text{где } |\delta| \text{ — малое приращение входных данных.}
```

Тогда вычисленный $n$-й член $\tilde{a}_n$:
```math
\tilde{a}_n = \tilde{a}_1 + (n-1)d = (a_1 + \delta) + (n-1)d = \underbrace{a_1 + (n-1)d}_{a_n} + \delta = a_n + \delta
```

Оценим погрешность выходных данных:
```math
|\tilde{a}_n - a_n| = |(a_n + \delta) - a_n| = |\delta|
```

Имеем **малое** приращение выходных данных, равное приращению входных.

$\Rightarrow$ Выполняется условие **(3)** из определения корректной задачи (устойчивость).

> **Вывод:** $\boxed{\text{Нахождение } n\text{-го члена арифметической прогрессии — корректная задача.}}$

---

### 2. Нахождение члена геометрической прогрессии

**Задача:**
Дан $b_1$ — первый член, $q$ — знаменатель ($q > 1$). Найти $b_n$.

**Дано:**
```math
b_n = b_1 \cdot q^{n-1}
```

**Анализ устойчивости:**
Внесем возмущение в $b_1$:
```math
\tilde{b}_1 = b_1 + \delta, \quad \text{где } |\delta| \text{ — малое приращение входных данных.}
```

Вычисленный член $\tilde{b}_n$:
```math
\tilde{b}_n = (\tilde{b}_1) \cdot q^{n-1} = (b_1 + \delta) \cdot q^{n-1} = b_1 \cdot q^{n-1} + \delta \cdot q^{n-1}
```

Оценим погрешность:
```math
|\tilde{b}_n - b_n| = |b_1 q^{n-1} + \delta q^{n-1} - b_1 q^{n-1}| = |\delta| \cdot |q|^{n-1}
```

**Вывод зависит от постановки:**
1.  Если $n$ — **фиксировано** (небольшое число), то задача условно **корректна**.
2.  Если $n$ — **любое** (стремится к бесконечности) и $|q| > 1$, то ошибка растет экспоненциально ($\delta \cdot q^{n-1} \to \infty$). Малому изменению входа соответствует *большое* изменение выхода.

> **Вывод:** $\boxed{\text{При больших } n \text{ задача некорректна (неустойчива).}}$

---

### 3. Нахождение кратного корня полинома

**Дано:**
Уравнение вида $(x-a)^n = 0$, где $n > 1$ (целое).
Входные данные — коэффициенты многочлена (в данном случае рассматриваем возмущение свободного члена/правой части).
Точный корень: $x = a$.

**Анализ устойчивости:**
Пусть свободный член изменился на малую величину $\delta$ (малое изменение входных данных).
Уравнение приняло вид:
```math
(\tilde{x} - a)^n = \delta
```

Выразим новый корень $\tilde{x}$:
```math
\tilde{x} - a = \sqrt[n]{\delta} \quad \Rightarrow \quad \tilde{x} = a + \sqrt[n]{\delta}
```

Оценим погрешность (разницу между корнями):
```math
|\tilde{x} - x| = |a + \sqrt[n]{\delta} - a| = |\sqrt[n]{\delta}|
```

Для малых $\delta$ (где $0 < \delta < 1$) и $n > 1$ справедливо неравенство:
```math
|\sqrt[n]{\delta}| \gg |\delta|
```
*(Например, если $\delta = 10^{-6}, n=2$, то ошибка $\sqrt{10^{-6}} = 10^{-3}$, что в 1000 раз больше возмущения).*

$\Rightarrow$ **НЕ** выполняется условие устойчивости (малому возмущению соответствует значительно большее изменение корня).

> **Вывод:** $\boxed{\text{Нахождение корней многочлена с кратным корнем — некорректная задача.}}$
